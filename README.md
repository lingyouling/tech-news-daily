# 科技新闻日报 📰

每天自动抓取 AI、科技、财经类新闻，生成可浏览的静态网站。

## 新闻源

| 来源 | 类型 | 说明 |
|------|------|------|
| [36氪](https://36kr.com) | 科技/财经 | 综合科技资讯 |
| [爱范儿](https://www.ifanr.com) | 科技 | 数字生活与产品 |
| [少数派](https://sspai.com) | 科技 | 效率工具与软件 |
| [蓝点网](https://www.landiannews.com) | 科技 | IT 资讯 |

## 自动更新

每天北京时间 **8:00** 通过 GitHub Actions 自动抓取并更新。

## 本地运行

```bash
npm install
npm run generate   # 抓取新闻并生成网页
npm run dev        # 本地预览（需 npx serve）
```
